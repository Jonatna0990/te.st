
Настройка БД: config->db.php
Миграции:
для выполнения миграции в командной строке необходимо перейти в каталог cd migrations и выполнить php task_create.php
- Создание таблицы users(php user_create.php)
- Добавление демо-данных в users(php user_demo_data.php)
- Создание таблицы orders(php order_create.php)
- Добавление демо-данных в orders(php order_demo_data.php)


Для обоих задач используется общая модель User
